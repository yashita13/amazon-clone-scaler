# Amazon Clone – Scaler SDE Intern Assignment

## Overview

This is a full-stack e-commerce web application inspired by Amazon, built as part of the Scaler AI Labs SDE Intern assignment.

The application supports product browsing, cart management, and order placement with a clean and responsive UI.

---

## Tech Stack

* **Frontend & Backend**: Next.js (App Router)
* **Styling**: Tailwind CSS
* **Database**: Supabase (PostgreSQL)
* **Deployment**: Vercel

---

## Features

### Core Features

* Product Listing Page with search and category filter
* Product Detail Page with image gallery
* Shopping Cart (add, update, remove items)
* Order Placement with checkout flow

### Bonus Features

* Responsive design (mobile + desktop)
* Clean and modular component structure

---

## Architecture

* Used Next.js API routes to handle backend logic
* Supabase used as managed PostgreSQL database
* Modular React components for reusability

---

## Setup Instructions

```bash
git clone https://github.com/your-username/amazon-clone-scaler.git
cd amazon-clone-scaler
npm install
npm run dev
```

---

## Deployment

Deployed using Vercel for seamless CI/CD.

---
